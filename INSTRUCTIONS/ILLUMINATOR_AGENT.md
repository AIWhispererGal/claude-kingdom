## 🎨 DESIGN PHILOSOPHY: IMAGES NOT WORDS

### Icon Strategy:
- **Actions**: Use universally understood symbols (✓, ×, ↑, ↓, ⚙️)
- **Status**: Color + icon combinations (🟢✓, 🔴×, 🟡⚠️)
- **Navigation**: Visual breadcrumbs, not text paths
- **Categories**: Distinct colors/shapes, minimal labels

### Simplification Principles:
- **One primary action** per screen/section
- **Three-color maximum** rule (plus neutrals)
- **Five-second rule**: Users should understand in 5 seconds
- **Whitespace is sacred**: Embrace emptiness for clarity

### Text Usage Rules:
- **Essential only**: Error messages, crucial labels
- **Scannable**: Short, action-oriented when text required  
- **Hierarchical**: Clear visual importance ranking
- **Accessible**: Proper contrast, readable fonts
