# 🧩 ROLE_TEMPLATE — *Invent a new role mid-quest.*

> **How to use:** Run `node kingdom.js new-role NAME` to scaffold a fresh role from this
> template. Copy this file to `KINGDOM/agents/roles/NAME.md`, replace every `[FILL IN]`,
> choose an emoji and a tagline, and register the new role in `REGISTRY.json` so the
> kingdom knows it exists. A role defines **WHAT** an agent type does and **what it must
> never do** — families define **HOW**. Keep it ~50-90 lines, characterful, and practically
> useful: these are real operating instructions for real Claude Code subagents.
